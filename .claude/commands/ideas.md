# /ideas — Deep vault scan for idea generation

Run a comprehensive idea generation scan across the entire vault. Cross-domain pattern detection, graph analysis, and creative synthesis.

## Steps

1. Read ALL daily notes in `Daily/`
2. Read all project context files in `Projects/`
3. Analyze the vault structure:
   - Which notes are orphaned (not linked to anything)?
   - Which notes are highly connected?
   - What tags or themes appear most frequently?
   - What links exist but lead to empty or missing notes?
4. Cross-reference domains — look for unexpected connections between:
   - Different projects (IMG Creator ↔ Ad Scraper)
   - Technical work ↔ personal reflections
   - Recurring themes across different time periods

## Output

Generate an **Idea Report** with these sections:

### Vault Structure Highlights
- Orphaned notes worth developing
- Highly connected themes
- Missing links that reveal latent interests

### What's Working
- Patterns that show momentum and progress

### Tools to Build
- Specific tools, features, or automations suggested by vault patterns

### Ideas to Explore
- Cross-domain connections
- Topics I keep circling but haven't committed to
- Things I should investigate based on vault trajectory

### Things to Write
- Notes, essays, or documentation the vault is missing

Keep each idea actionable. Not "you should think about X" but "build X because Y evidence from the vault."
