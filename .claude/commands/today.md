# /today — Morning review

Pull together everything relevant to plan my day. Read recent daily notes, check what's in progress, and create a prioritized plan.

## Steps

1. Read the last 7 daily notes from `Daily/`
2. Read all project context files from `Projects/`
3. Look at incomplete tasks (`- [ ]`) across recent daily notes
4. Check what I was focused on yesterday and the pattern over the past week

## Output

Create (or update) today's daily note in `Daily/` with the format:

```
# [today's date YYYY-MM-DD]

## Focus
- [The #1 thing I should focus on today based on recent momentum]

## Carry-over tasks
- [ ] [incomplete tasks from recent days that still matter]

## Notes

## Tasks
- [ ]
```

Then give me a brief spoken summary: what's hot, what's stale, what deserves attention today based on what the vault says I've been working on.
